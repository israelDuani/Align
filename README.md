### Quick start
You can either run the backend manualy on your machine and create mongo service running, or you can just use docker :smile:

##### First time build / any changes on code
#
```sh
docker-compose build --no-cache
```

##### Run the project
#
```sh
docker-compose up -d
```

##### Stop the project
#
```sh
docker-compose down
```
